Welcome

Part of becoming a health data scientist is asking questions and gathering data to enable you to answer these questions effectively.

Learning Objectives

In this chapter, we would discuss the process of using reproducible scripts in R to work with data to answer our research questions. You would learn how to:

  • Import data from Excel and CSV files
  • Load built in data sets
  • Create a data table
  • Understand the basic data types

Introduction

Where do we get data to answer our research questions? As a health professional, you might conduct an experiment and record your observations or you may send out questionnaires and record the responses the participants gave.

This data may be stored in different formats, as .csv (comma separated values), excel files (.xlsx or .xls) and many more. R has functionality to work with many different file formats.
